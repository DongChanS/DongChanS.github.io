## 2) Callback function



### 2-1. Callback function 이란?

```javascript
const orderCoffee = (order, callback) => {
    setTimeout(() => {
        let coffee = order
        callback(coffee)
    }, 1000);
}

console.log(orderCoffee('아아', (coffee) => {
    console.log(coffee)
}))
```

callback function의 장점 : 비동기 조작의 순서는 확실하게 제어할 수 있다!

(이것은 call stack과는 거의 독자적으로 돌아가는 형태이긴함.)
